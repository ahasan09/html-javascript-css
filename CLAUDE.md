# HTML JavaScript CSS

Collection of 7 standalone static mini-projects demonstrating HTML, CSS, and vanilla JavaScript techniques.

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Project Structure
```
html-javascript-css/
├── blog/
├── restaurant-menu/
├── flexbox-cards/
└── <other-projects>/
```
Each subfolder is a self-contained static site.

## Development
No installation required. Open any project's `index.html` directly in a browser:
```bash
open html-javascript-css/<project>/index.html
```

## Key Notes
- No build step, no dependencies — pure static files.
