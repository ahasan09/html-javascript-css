# HTML, JavaScript & CSS Projects

A collection of beginner-friendly mini-projects demonstrating core web development concepts using plain HTML, CSS, and JavaScript — no frameworks required.

## Projects

| Folder | Description |
|--------|-------------|
| `array-operation` | Interactive demo of common JavaScript array methods |
| `flexbox-cards` | Responsive card layout built with CSS Flexbox |
| `image-positioning` | CSS techniques for positioning and aligning images |
| `my-blog` | Simple static blog layout with HTML & CSS |
| `resturant-menu` | Restaurant menu UI with styled item cards |
| `social-mockup` | Social media profile page mockup |
| `string-operation` | Interactive demo of JavaScript string methods |

## Running the Projects

No build tools or installation needed. Just open any `index.html` in a browser:

```bash
# macOS
open flexbox-cards/index.html

# Linux
xdg-open flexbox-cards/index.html

# Windows
start flexbox-cards/index.html
```

Or use a local server for the best experience:

```bash
npx serve .
# then navigate to the subfolder in the browser
```

## Prerequisites

- Any modern web browser (Chrome, Firefox, Edge, Safari)
